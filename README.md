# Лучший в мире виджет для банка!

## Функции
Кодирование банковских реквизитов - Функция возвращает строку с замаскированным номером. Для карт и счетов используются разные типы маскировки.
Фильтрация банковских реквизитов - Функция возвращает новый список словарей, содержащий только те словари, у которых ключ 
state  соответствует указанному значению.
Сортировка банковских реквизитов - Функция  возвращает новый список словарей, отсортированный по дате (date).
